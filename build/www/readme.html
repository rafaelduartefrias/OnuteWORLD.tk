What the hell is this?

BonziWORLD is a node.js and socket.io based chat client featuring everyone's most hated loved rainbow gorilla. The client also uses speak.js to provide text-to-speech voices. Not exactly the original BonziBUDDY voice, but hey, it works. The voice is using eSpeak, but it will crash almost any longer text.

Why did you make this? Why WOULD you make this?

Memes. Also because I'd already finished the BonziBUDDY portion of this project like a day/week/month in advance and was trying to find some way to put it to use.

How do I use it?

  1. Enter a nickname (under 4096 characters). If you don't enter one, you'll be named "User". (we r legion du nut 4 get). On private rooms have more 8192 chars (8kb).
  2. Optionally, enter a room ID. If you want to join someone else, their room ID will be in the bottom-right corner of the screen.
    • If you enter a room ID that doesn't exist, you will be placed in a private room which can only be joined by people you share the ID with. Copy the private link and share other.
  3. Be a BonziBUDDY.
